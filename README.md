# Land
领地系统
功能:
  - 自定义圈地提示
  - 自定义玩家领地上限
  - 自定义禁止圈地世界
  - 自定义购买领地价格
  - 领地成员独立权限
  - 立体领地
  - GUI 操作
  - 领地套娃?(主领地内可设置数个子领地(配置可更改))
  - 领地粒子区域显示
  - 领地出售
  - 领地赠送
  - 领地寄售
  - 自定义经济核心
  - 领地GUI查找
  - 使用工具圈地

领地权限:
   - 禁止破坏
   - 禁止放置 
   - 禁止传送
   - 禁止进入
   - 禁止丢弃物品
   - 禁止使用箱子 熔炉 
   - 禁止PVP
   - 禁止伤害生物
   - 使用桶
   - 使用打火石
   - 创建子领地
   - 使用展示框


使用方法:
  安装前置 经济核心
  插件指令: /la help
    ① 玩家输指令/la pos <1/2> 确定两点
    ② 玩家手持圈地工具(默认木锄) 点击方块进行圈地

之后执行 /la create 名称 即可创建领地 (领地范围为长方体)
